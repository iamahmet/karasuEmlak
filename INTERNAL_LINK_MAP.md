## Cross-Linking Strategy

### Rules:
1. **Contextual Only**: Links must be contextually relevant
2. **No Exact-Match Spam**: Use natural language variations
3. **Comparative Language**: Use comparison context when linking between hubs
4. **Bidirectional**: Both hubs link to each other

### Implementation:

#### Karasu → Kocaali Links:
- **Location**: Sidebar "İlgili Sayfalar" section
- **Anchor Text**: "Kocaali'de Satılık Ev Seçenekleri →"
- **Context**: Alternative option for buyers
- **Page**: `/karasu-satilik-ev`

#### Kocaali → Karasu Links:
- **Location**: Comparison section
- **Anchor Text**: "Karasu Satılık Ev Rehberi"
- **Context**: Comparison and alternative
- **Page**: `/kocaali-satilik-ev`

## Internal Linking Hierarchy

### Level 1: Blog → Local Hub → Main Pillar
```
Blog Post → Cornerstone → Pillar
Example: 
  Blog: "Karasu'da Ev Alırken Yapılan Hatalar"
  → Links to: /karasu-satilik-ev (pillar)
  → Links to: /karasu-merkez-satilik-ev (cornerstone)
```

### Level 2: Neighborhood → Cornerstone → Pillar
```
Neighborhood Page → Cornerstone → Pillar
Example:
  /mahalle/merkez → /karasu-merkez-satilik-ev → /karasu-satilik-ev
```

### Level 3: Hub ↔ Hub (Cross-Linking)
```
Karasu Pillar ↔ Kocaali Pillar
Example:
  /karasu-satilik-ev → /kocaali-satilik-ev (contextual comparison)
```

## Anchor Text Rules

### DO:
- ✅ Natural language variations
- ✅ Contextual phrases
- ✅ Question-based anchors
- ✅ Descriptive text

### DON'T:
- ❌ Exact-match keyword stuffing
- ❌ Repeated identical anchors
- ❌ Over-optimized text
- ❌ Unnatural language

## Q&A Distribution Strategy

### Q&A Categories:
1. **Bilgi** (Information): "nedir, nasıl, neden"
2. **Karşılaştırma** (Comparison): "X mi Y mi"
3. **Karar Verme** (Decision): "mantıklı mı, uygun mu"
4. **Risk** (Risk): "dikkat edilmesi gerekenler"
5. **Yatırım** (Investment): "yatırım perspektifi"

### Distribution:
- **Pillar Pages**: High-priority Q&As (all categories)
- **Cornerstone Pages**: Medium-priority Q&As (relevant categories)
- **Blog Posts**: Low-priority Q&As (topic-specific)
- **Neighborhood Pages**: Local Q&As (location-specific)

## News Integration

### News → Hub Linking:
- News articles link to relevant hub pages
- Real estate angle added to all news

→ Links to: /karasu-satilik-ev

→ Links to: /mahalle/[relevant-neighborhood]
